能的，而且这是 Java 最基本的用法之一。
你可以把它理解成 “一个类去用另一个类里的方法/变量”，就像 C++ 调用另一个类一样。

1. 同一个包里
如果两个类在同一个包（默认就是同一个文件夹）里，并且类的访问修饰符允许（比如是 public 或者包内可见），直接用就行：

java
复制代码
// File: Tool.java
public class Tool {
    public void sayHello() {
        System.out.println("Hello from Tool!");
    }
}

// File: Main.java
public class Main {
    public static void main(String[] args) {
        Tool t = new Tool(); // 创建 Tool 类对象
        t.sayHello();        // 调用 Tool 类方法
    }
}
编译运行：

bash
复制代码
javac Tool.java Main.java
java Main
2. 不同包里
如果两个类不在同一个包里，你就需要：

用 package 声明包名

用 import 导入类

还得保证访问修饰符允许（public）

例子：

java
复制代码
// File: utils/Tool.java
package utils; // 声明包名

public class Tool {
    public void sayHello() {
        System.out.println("Hello from Tool in utils package!");
    }
}
java
复制代码
// File: Main.java
import utils.Tool; // 导入 Tool 类

public class Main {
    public static void main(String[] args) {
        Tool t = new Tool();
        t.sayHello();
    }
}
编译：

bash
复制代码
javac utils/Tool.java Main.java
java Main
✅ 核心规则：

访问修饰符（public, protected, default, private）会限制能不能用。

public 类必须放在和它同名的 .java 文件里。

同一个包不用 import，不同包必须 import。
